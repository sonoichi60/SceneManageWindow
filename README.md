SceneManageWindow
======================
プロジェクト内のシーンを自動取得・一覧表示し、いろいろできる拡張機能です。

導入方法
-----
1.[ここ](https://drive.google.com/uc?id=1QcYkvaYXyOyIWzuvfs3V6pLmusk9mkgM)からパッケージをダウンロード

2.パッケージをインポート

3.Window -> SceneManageWindow から開く

以上です。
